# SQL-Project

## Description
 This project contains SQL queries for an online shopping database system.

## Features
 - Create  product table
 - Insert records into tables
 - Retrieve data using SELECT queries
 - Update and delete data
 - Drop table
 - Generate output using SPOOL

## Tables Structure
 The table contains :
 - Product_ID
 - Product_Name
 - Price
 - Stock

## Tools used
 - SQL Plus
 - GitHub
